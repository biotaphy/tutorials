Resolve taxonomy in a Darwin Core Archive (DwCA)

Request a DwCA from iDigBio

Request a DwCA from iDigBio using their iDigBio Download API.
Example: Euphorbia

The response status via the resulting status_url. When status response task_status == SUCCESS, pull via included download_url.

# pull text response directly to terminal using curl
curl <query_url>
curl <status_url>
# save response into file with wget 
wget <download_url>
