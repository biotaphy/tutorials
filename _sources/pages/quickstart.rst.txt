===============
Quick Start
===============

-----------------------
Install Docker
-----------------------

Download by following instructions for your operating system at
`Get Docker <https://docs.docker.com/get-docker/>`_ , then install `Docker
<https://docs.docker.com/get-started/>`_.

-----------------------
Download Tutorial
-----------------------

The `Biotaphy tutorial <https://github.com/biotaphy/tutorials>`_  can be installed by
downloading from Github.  This code repository contains scripts, Docker composition
files, and data for the tutorials.

-----------------------
Initiate each tutorial
-----------------------

Choose a command to initiate with the default data.  Commands and example configuration
files referencing test data are in the Hands-on section of each tutorial.  Run the
tutorial script `run_tutorial` with no parameters to get a list of valid commands and
required options.

.. code-block::

       ./run_tutorial.sh

-----------------------
Examine results
-----------------------

All outputs from tutorial exercises are copied to the local system in a newly
created `output` directory under the downloaded `tutorials/data/` directory.
